# Welcome to MoodleNet styleguide
The current styleguide aims to provide a solid foundation to build, extend and customize the MoodleNet experience. 
The main goal is to produce a set of primitive and aggregate UI components, that will act and appear consistent with Moodle visual identity and MoodleNet system design language.


Each MoodleNet component is:
- stateless
- written in `typescript`
- styled with [styled-components](https://www.styled-components.com/) 
- built upon [rebass](https://rebassjs.org) primitives.
- themed

---

This styleguide version reflects the [MoodleNet clickable prototype](https://projects.invisionapp.com/d/main?origin=v7#/console/18048686/374475280/preview) released in july 2019 and explained in [this blog post](https://blog.moodle.net/2019/clickable-prototype/).
Mockups are built in sketch, and are available to download.


## Spacing
MoodleNet uses a `8px` multiplier to manage margins and paddings to ensure consistence across the whole application.
